==What is this mod?==

This mod is made for Hypixel's Skyblock Gamemode. It provides information about all the custom skyblock items, such as recipe, usages, price and an ingame wiki.

==Controls== R - Show crafting recipe of item.
U - Show crafting uses for item.
F - Toggle item as favourite.
L - Add item to inventory (creative-only).
Double-Click Search Bar - Toggle item search mode
Middle-Click Item - Quick search for item
*These controls work on items in your inventory too!
