==What is this mod?==

This mod is made for Hypixel's Skyblock Gamemode. The mod is focused on providing a variety of unique features not found in other mods. See below for a list of (major) features.

==Features== - Skyblock Item List
- Item Prices (Avg/Lowest BIN)
- Custom Auction House (/neuah)
- Item Collection Log (/neucl)
- QuickCommandsâ„¢ (above search bar)
- Dungeon Chest Profit Checker
- Much more to come!
==Api==

Some features require the use of an api key. (~15 req/min) If you type /api new, the key should be automatically detected. If you want to manually insert a key, use the options gui.

==Controls== R - Show crafting recipe of item.
U - Show crafting uses for item.
F - Toggle item as favourite.
L - Add item to inventory (creative-only).
Double-Click Search Bar - Toggle item search mode
Middle-Click Item - Quick search for item
*These controls work on items in your inventory too!

==Boolean Searching== The search bar supports boolean searching. Try using | (or), & (and) and ! (not). ==Any Problems?==
